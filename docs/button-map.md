# Button Map

Confirmed G29 button mappings from real packet captures.

## Byte 0

- `X`
  - `0x08 -> 0x18`
- `O`
  - `0x08 -> 0x48`
- `T`
  - `0x08 -> 0x88`
- `S`
  - `0x08 -> 0x28`
- `up`
  - `0x08 -> 0x00`
- `left`
  - `0x08 -> 0x06`
- `down`
  - `0x08 -> 0x04`
- `right`
  - `0x08 -> 0x02`

## Byte 1

- `right_paddle`
  - `0x00 -> 0x01`
- `left_paddle`
  - `0x00 -> 0x02`
- `L2`
  - `0x00 -> 0x08`
- `L3`
  - `0x00 -> 0x80`
- `R2`
  - `0x00 -> 0x04`
- `R3`
  - `0x00 -> 0x40`
- `Share`
  - `0x00 -> 0x10`
- `Options`
  - `0x00 -> 0x20`
- `left_paddle + right_paddle`
  - `0x00 -> 0x03`

## Byte 2

- `+`
  - `0x00 -> 0x80`

## Byte 3

- `-`
  - `0x00 -> 0x01`
- `back`
  - `0x00 -> 0x08`
- `PS`
  - `0x00 -> 0x10`
- `dial_right`
  - pulse: `0x00 -> 0x02 -> 0x00`
- `dial_left`
  - pulse: `0x00 -> 0x04 -> 0x00`
